Steps to configure the java web application

 1. Import the project in spring.
 2. Change the project name.
 	> pom.xml
 	> web.xml
 	> file name of resources
 		>database
 		>properties
 		>security
 		>servlet

 3. Rename all the src/main/java
 	>bean
 	>common
 	>controller
 	>dao
 	>model
 	>service
 	>utils		


  4. change "defaultjavawebapp" your system name	
  <path>/defaultjavawebapp</path>
  <finalName>defaultjavawebapp</finalName>

  5. Set the web project settings in to your name.
  	>Delete the project
  	mvn eclipse:eclipse -Dwtpversion=2.0
  	> clean, install and test maven.
